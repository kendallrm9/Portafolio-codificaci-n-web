EJEMPLOS DE ATRIBUTOS GLOBALES

Los atributos globales son atributos comunes a todos los elementos HTML; pueden usarse en todos los elementos, aunque pueden no tener efecto en alguno de ellos.

Los atributos globales pueden especificarse en todos los elementos HTML, e incluso en aquellos no especificados en el estándar. Esto significa que cualquier elemento no están también debe permitir estos atributos, aunque el uso de tales elementos hoy significa que el documento ya no cumple con HTML5.

accesskey

Proporciona y genera un acceso de teclado para el elemento actual.

class

Expecifica uno o más nombres de clases de un elemento ( ejemplo los subtitulos color red y font-size 15px).

dir

Especifica la dirección del texto para el contenido de un elemento, observe que le subtitulo dir esta a la derecha

hidden

Expecifica que un elemento no es todavía, o ya no es, relevante, es decir lo oculta, observe que el subtitulo hidden esta oculto.

id

Especifica un identificador único para un elemento ejemplo ver título con color blue y font-size 20px

lang

Especifica el lenguaje del contenido del elemento ejemplo este párrafo en español

style

Especifica un línea de estilo CSS de un elemento ejemplo este párrafo con style color verde

tabindex

Expecifica el orden de tabulación de un elemento. El orden comienza en uno.

tabindex

Expecifica el orden de tabulación de un elemento. El orden comienza en dos.

title

Especifica información adicional acerca de un elemento. La información generalmente será mostrada por un globo al poner el puntero, por ejemplo al ubicarse sobre este subtitulo aparecerá el mensaje "Ejemplo de título"

contenteditable

Especifica si el contenido de un elemento es editable o no, por ejemplo al dar clic sobre este texto se podrá editar.

draggable

Especifica si un elemento se puede arrastrar o no, este párrafo se puede arrastrar.

spellcheck

Especifica si el elemento ha de tener su ortografia y gramática marcada o no. Los elementos que pueden tener revisión ortográfica son: inputs de tipo texto, textareas y elementos en modo editable, por ejemplo al dar clic sobre este párrafo se pueden ver marcas ortográfica
